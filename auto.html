다시 넣기

줄을 그은 이 할 일이 마지막 페이지의 맨 끝에 다시 들어갑니다.
이름을 바꿔서 넣을 수도 있어요.

오토포커스 사용법

마크 포스터의 Autofocus 시스템을 따르는 종이 노트식 할 일 앱입니다.

기본 규칙

  • 떠오르는 할 일은 전부 추가하세요. 항상 마지막 페이지 끝에 들어갑니다. (한 페이지 10줄)
  • 페이지를 처음부터 훑고, 억지로 고르지 말고 "끌리는" 항목을 선택해 일합니다.
  • 하고 싶은 만큼만 하고 멈추세요. 조금씩 자주가 원칙입니다.
  • 끝 — 완료. 줄이 그어집니다.
  • 계속 — 줄을 긋고 마지막 페이지 끝에 다시 넣습니다. 이어서 할 일, 후속 조치, 반복 업무에 쓰세요.
  • 미결 — 노란 형광펜. 페이지가 끝나면 '비활성 할 일'로 옮겨집니다.
  • 페이지의 모든 항목이 처리되면 그 페이지는 자동으로 사라집니다.
  • 어떤 페이지를 처음 훑었는데 아무것도 끌리지 않으면, 하단의 "이 페이지 통째로 기각"으로 전부 미결 처리하세요. (마지막 페이지에는 적용하지 않습니다)
  • 비활성 할 일은 시간을 두고 다시 보세요. 정말 필요하면 활성으로 되살리되, 쪼개거나 표현을 바꾸는 게 좋습니다.

데이터

  • 이 기기(브라우저)에 자동 저장됩니다.
  • Safari로 열고 공유 → 홈 화면에 추가를 하면 앱처럼 쓸 수 있고 데이터가 유지됩니다.
  • 기기를 바꾸거나 백업이 필요하면 내보내기 / 가져오기를 이용하세요.
